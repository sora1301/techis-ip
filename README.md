## 商品管理システム

### 概要

このシステムでは、店舗で商品の在庫管理を行うことができます。
商品の新規登録、削除を行うことができます。

## 主な機能
- ユーザー登録
- ログイン・ログアウト機能
- 商品一覧画面
- 商品新規登録、削除機能

## 開発環境
php 
MySQL 
Laravel
